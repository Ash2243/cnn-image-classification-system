# 🧠 CNN Image Classification System

## 🚀 Project Overview

This project focuses on building a Convolutional Neural Network (CNN) based image classification system using TensorFlow and Keras.  
The model is trained to classify images into categories using deep learning techniques with preprocessing, augmentation, and performance evaluation.

---

## 🎯 Objectives

- Build an end-to-end image classification pipeline
- Preprocess image data for deep learning
- Apply data augmentation for better generalization
- Train a CNN model for accurate predictions
- Evaluate model performance on unseen data
- Predict custom images using the trained model
